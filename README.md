#### A project I'm proud of:
• I built a **355M parameter GPT-style Transformer** from **scratch** using PyTorch, which I'm particularly proud of because it required understanding the architecture deeply rather than relying on existing libraries.
<br>
<br>• I wanted to truly understand how large language models work under the hood — not just use APIs, but implement every component myself.
<br>
<br>• I implemented the full decoder-only Transformer architecture including **Multi-Head Self-Attention**, **Causal Masking**, **Layer Normalization**, and **GeLU activations**. I also wrote a **custom Byte Pair Encoding (BPE) tokenizer** from scratch and built an optimized data pipeline for large-scale training. To improve memory efficiency and convergence, I applied **weight tying** across the embedding and output layers. Finally, I wrapped everything in an interactive text generation interface using **Chainlit** so the model could be tested in real time.
